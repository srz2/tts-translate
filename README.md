TTS Translate
=============

This was a tutorial example from [Traversy Media](https://www.youtube.com/watch?v=V0P3Opf-zUs)
where he creates a simple Text to Talk Speech application using Google Translate API.

## Technologies

- Vanilla Javascript
- Google Translate API
- Bootstrap
